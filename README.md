# toop-parent-pom

Shared Maven parent POM.
The latest version is **1.0.3**.

# How to use it
Include the following code in you pom.xml:

```xml
<parent>
  <groupId>eu.toop</groupId>
  <artifactId>parent-pom</artifactId>
  <version>1.0.3</version>
</parent>
```
